# Sport Challenge Bot v6 — несколько групп

Один Telegram-бот обслуживает несколько независимых групп.

## Что изменилось

- каждая группа автоматически регистрируется после `/start`;
- у каждой группы свой активный челлендж, рейтинг и статистика;
- пользователь может состоять сразу в нескольких группах;
- результаты разделены по челленджам групп;
- в личном чате появилась кнопка `🏠 Выбрать группу`;
- вход в группу выполняется по кнопке `➕ Вступить в челлендж`;
- администратор управляет только выбранной группой;
- старая база мигрируется автоматически без удаления результатов.

## Установка

Замените файлы текущего проекта содержимым этого архива и отправьте изменения в GitHub. Railway выполнит deploy автоматически.

Переменные:

```env
BOT_TOKEN=...
DATABASE_PATH=/data/sport_challenge.db
TIMEZONE=Europe/Moscow
ADMIN_IDS=123456789
YESTERDAY_EDIT_UNTIL=12:00
```

Volume Railway и `/data` не удалять.

## Как добавить новую группу

1. Добавьте того же бота в новый групповой чат.
2. Разрешите боту отправлять сообщения.
3. Отправьте в группе `/start`.
4. Администратор и участники нажимают `➕ Вступить в челлендж`.
5. Администратор в личном чате выбирает группу → `🛠 Управление группой` → `➕ Новый челлендж`.

В каждой группе кнопки `🏆 Общий рейтинг` и `📊 Статистика челленджа` используют только данные этой группы.

## Миграция старой группы

Существующие челленджи v5 будут помещены в техническую группу `Старая группа`. Чтобы продолжить работу в реальном чате, создайте новый челлендж в нужной группе. Старые результаты не удаляются.
