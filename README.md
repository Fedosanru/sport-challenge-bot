# Sport Challenge Bot v2

Во второй версии реализованы все 10 улучшений:

1. вечерние личные напоминания;
2. список участников, не внёсших результат сегодня;
3. просмотр результатов текущего дня;
4. ввод за вчера до заданного времени;
5. подтверждение результата перед сохранением;
6. текущая и максимальная серии без пропусков;
7. недельный рейтинг и автоматическая публикация;
8. выгрузка в Excel;
9. кнопочная админ-панель;
10. итоговые номинации.

## Обновление со старой версии без потери результатов

1. Остановите старого бота.
2. Обязательно сохраните копию `sport_challenge.db`.
3. Замените `bot.py`, `db.py` и `requirements.txt` файлами из новой версии.
4. Обновите зависимости:

```bash
source .venv/bin/activate
pip install -r requirements.txt
```

5. Добавьте в `.env`:

```env
REMINDER_TIME=20:00
YESTERDAY_EDIT_UNTIL=12:00
```

6. Запустите бота:

```bash
python bot.py
```

При первом запуске база автоматически обновится. Существующие пользователи,
челленджи и результаты сохранятся.

## Настройка `.env`

```env
BOT_TOKEN=токен_из_BotFather
ADMIN_IDS=123456789
RESULTS_CHAT_ID=-1001234567890
TIMEZONE=Europe/Moscow
DATABASE_PATH=/data/sport_challenge.db
REMINDER_TIME=20:00
YESTERDAY_EDIT_UNTIL=12:00
```

На Railway база должна находиться на подключённом Volume, например `/data`.

## Как пользоваться

Каждый участник открывает личный чат с ботом и отправляет `/start`.

Главное меню:

- `📝 Внести результат`
- `📊 Моя статистика`
- `🏆 Общий рейтинг`
- `📅 Результаты по дням`
- `🏅 Рейтинг недели`
- `🔔 Напоминания`
- `ℹ️ Правила`

Администратор открывает `🛠 Админ-панель` и может:

- создать челлендж пошагово;
- завершить челлендж;
- посмотреть, кто не внёс результат;
- скачать Excel;
- показать недельный рейтинг;
- показать итоговые номинации.

## Важные особенности

- Напоминание отправляется один раз в день после `REMINDER_TIME`.
- Участник может самостоятельно отключить напоминания.
- За вчера можно внести результат только до `YESTERDAY_EDIT_UNTIL`.
- Повторная запись за тот же день заменяет предыдущую.
- Еженедельный рейтинг публикуется после 7-го, 14-го и 21-го дня.
- Автоматические действия выполняются только пока бот постоянно запущен.
- Excel содержит листы: общий рейтинг, результаты по дням, участники.

## Режим общего чата

Общий Telegram-чат работает только как информационное табло. Бот публикует там недельные и итоговые рейтинги. Ввод результатов, статистика, настройки, напоминания, Excel и админ-панель доступны только в личном чате.

При обращении в группе бот показывает кнопку «Открыть личный чат с ботом». Команда `/id` остаётся доступной для получения `RESULTS_CHAT_ID`.

Рекомендуется включить Privacy Mode через BotFather:

```text
/mybots → выбрать бота → Bot Settings → Group Privacy → Turn on
```


## Возможности версии v4

- Красивый общий и недельный рейтинг с прогресс-барами, отрывом от лидера, идеальными днями и сериями.
- Достижения сохраняются в SQLite и выдаются только один раз.
- Кнопка «Мои достижения» показывает коллекцию наград пользователя.
- После сохранения результата «Умный тренер» анализирует динамику, ближайшую дневную цель, отрыв до следующего места и близкие достижения.
- Внешний AI API не нужен: анализ выполняется локально и бесплатно.
